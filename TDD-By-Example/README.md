### **Chapter 3. 모두를 위한 평등**

### **Chapter 12. 드디어, 더하기**

환율을 계산해 가며 통화를 더하는 것은 복잡한 일이다.
간단한 달러간의 덧셈을 시작으로 구현해보자.

**가장 어려운 제약은 다중 통화 사용에 대한 내용을 시스템의 나머지 코드에게 숨기고 싶다는 점이다.**
한 가지 전략은 참조통화를 사용하는 것, 하지만여러 환율을 쓰기가 쉽지 않다.
하지만 산술 연산처럼 다룰 수 있다면 편하게 여러 환율을 표현할 수 있을 것 같다.
산술 연산에 관한 Money와 비슷하게 동작하지만 사실은 두 Money의 합을 나타내는 새로운 객체 *(객체와 외부 프로토콜이 같으면서 내부 구현은 다른 새로운 객체-imposter)* 를 생성하여
연산의 결과로 Sum과 같은 여러 Expression들을 생성하고, 환율을 이용해 결과 Expression을 단일 통화로 축약할 수 있다.


Check list
- [ ] $5 + 10CHF = $10(환율이 2:1)
- [ ] $5 + $5 = $10

지금까지 한 작업

- 우리에게 필요한 계산에 대한 가능한 메타포들을 신중히 생각해봤다(Bank, Expression)
- 새 메타포에 기반하여 기존의 테스트를 재작성했다.
- 테스트를 빠르게 컴파일했다.
- 그리고 테스트를 실행했다.
- 진짜 구현을 만들기 위해 필요한 리팩토링을 약간의 전율과 함께 기대했다.
